## 四.数学公式

### 数学公式latex语法

使用`$`表示,其中一个\$表示在行内, 两个 \$ 表示独占一行.

例如质量守恒公式: $$E=mc^2$$
```
$$E=mc^2$$
```

支持**LaTeX**编辑显示支持, 例如$\sum_{i=1}^n a_i=0$,
```
$\sum_{i=1}^n a_i=0$
```
